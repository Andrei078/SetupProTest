🎮 SetupProTest

SetupProTest este o aplicație web modernă pentru testarea perifericelor tale: tastatură, mouse, controller, webcam și audio — toate într-un singur loc.

---

🚀 Funcționalități

- ⌨️ Test tastatură (input + latență)
- 🖱️ Test mouse (tracking + click)
- 🎮 Test controller (Xbox / PS4 / PS5)
- 🎧 Test audio (stânga/dreapta + sunet)
- 📷 Test webcam (video live)
- ⚙️ Setări personalizate
- 📊 Statistici în timp real (latență, drift, presiune)

---

🧠 Tehnologii folosite

- HTML5
- CSS3 (design modern + efecte vizuale)
- JavaScript (Gamepad API, Media API)
- Responsive Design

- Dă click pe pagină înainte de utilizare
- Rulează pe "localhost"
